Methodology

This project deals with the cryptanalysis of ciphers and hashed values. In hashed value cryptanalysis, user enters the hashed value and specifies the choice of hashed value. The machine then implements cryptanalysis using dictionary attack with a dictionary of curated 1,430,894 common passwords and words.

In the case of Caeser cipher, user enters the choice of language from the options given, and then enters the encrypted value. The machine then outputs the most probable key and plaintext combination instead of printing every single combination.



Assumptions

1: Hashed value entered is correct and in the appropriate format

2: For Caeser cipher cryptanalysis, the correct language is chosen (English, Spanish, or German)

3: For SHAKE256, hexdigest parameter=100

4: In Caeser cipher, string must be at least 3 characters



List of cipher decryptions available

1. MD5

2. SHA1

3. SHA256

4. SHA512

5. SHAKE256

6. Caeser cipher (English)

7. Caeser cipher (Spanish)

8. Caeser cipher (German)
